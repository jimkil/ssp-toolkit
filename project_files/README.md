# SSP Automation

This repository contains the files for managing and automating the System Security Plan (SSP) for our
projects. The SSP is a critical document that outlines the security controls and measures implemented
to protect our systems. The files in this repository are generated and maintained through automated
processes using the [CivicActions SSP Toolkit](https://github.com/CivicActions/ssp-toolkit).
